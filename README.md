## Descrição do Projeto
Este projeto consiste no desenvolvimento de uma ferramenta de "Document Intelligence" que permite o carregamento de ficheiros (PDFs, imagens de documentos, etc.) e a interação com os mesmos através de um chatbot. O sistema utiliza técnicas de OCR para extração de texto e RAG (Retrieval-Augmented Generation) para responder a perguntas baseadas no conteúdo dos documentos.

**Track:** B (AI Agents / Knowledge)
